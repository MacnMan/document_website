# MacSync LORS X1 LoRaWAN Rain Sensor Datasheet

<h2 className="device-section-title">
  ## Product Overview
</h2>

**MacSync LORS X1** is a high-performance **LoRaWAN® rain sensor** designed for accurate rainfall detection, real-time precipitation monitoring, and outdoor weather intelligence applications. Built using advanced solid-state optical sensing technology, the sensor delivers highly reliable rain detection without any mechanical moving parts, enabling maintenance-free operation in harsh outdoor environments.

Engineered for smart agriculture, flood monitoring, industrial automation, environmental monitoring, and smart-city weather stations, **MacSync LORS X1** accurately detects drizzle, moderate rainfall, and heavy rain intensity with fast response time and stable long-term performance.

With ultra-long-range **LoRaWAN® connectivity**, rugged IP-rated housing, intelligent ambient-light filtering, and ultra-low-power architecture, the device is ideal for remote field deployment across farms, reservoirs, highways, riverbanks, weather stations, irrigation systems, and industrial sites.

The sensor supports OTA configuration through the **Maya App**, onboard data buffering, real-time rainfall alerts, and seamless integration with TTN, ChirpStack, MQTT servers, and private LoRaWAN® networks — making **MacSync LORS X1** a powerful industrial IoT rain sensor for modern rainfall monitoring systems.
